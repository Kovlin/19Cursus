Templates

Créer des templates :

  • On fait un fichier html template de base (par ex base.html) qu'on met dans le repertoire templates de l'app'
  • On y inclus des blocs comme

    {% block mon_bloc %}

    du html...

    {% endblock %}

  • On inclus dans les templates à construire sur cette base {% extends 'base.html' %}
  • On configures une route vers le template à construire dans le fichier urls.py de l'app' qui pointe sur une vue
  • dans cette vue, on fait appel à la fonction render() en lui passant le template en parametre...
Ensuite, dans ce template à construire on inclus les différents blocs à réécrire du template de base en utilisant par exemple :
  • la réécriture d'un bloc en redefinissant

    {% block mon_bloc %}

    Le html qui remplacera celui du template de base

    {% endblock %}

  • on peut y inclure des variables avec {{ ma_variable }}
  • des boucles avec {% for xx in yy %}{% end for %}
  • des conditions avec {% if mon_parametre == 2 %}{% endif %}
  • de l'inclusion d'un autre template avec {% include 'autre/template.html' %}
