Les templates en mode statique :

Comment les créer :

  • On crée un fichier qui servira de template qu'on place dans les templates de l'app
  • On configure le path dans urls.py de l'app pour qu'il pointe vers la fonction qui permmettra de l'afficher dans views.py
  • On appelle la fonction render dans views.py avec les paramètres request et le chemin vers le html pour afficher la page
